    So why do people use bitcoin?

    Distrust of banks to keep their money secure; many have more faith in Bitcoin's blockchain system. The Blockchain & verification process makes it extremely difficult for counterfeiting to occur. Some like the partial anonymity offered by Bitcoin (some people buy VPNs with bitcoin cryptocurrency because they're worried about legal trouble if their name is attached to the payment).

    How do trasactions work?

    Each user has a Bitcoin wallet and a two-key address to sign transactions
    Public key --> anyone can look up, similar to e-mail address
    Private key --> used to send bitcoin, similar to e-mail password
    A Bitcoin address is an identifier of 26-35 letters and numbers
    Transactions are verified by network nodes through the use of cryptography
    Exchanges are recorded in a shared public ledger, called blockchain
    Collected in blocks, which represents an update of the user's balances on the network
    Transactions are confirmed by network within 10 minutes

    Block Components

  • Timestamp: The time when the block was found.
  • Refrence to Parent (Prev Hash): a hash of the previous block header which ties each block to its parent
  • Merkle Root (Tx_Root): a reduced representation of the set of transactions that is confirmed with this block
    The transactions are provided independently, and they form the body of the block
    There must be at least one transaction: The Coinbase.
    The coinbase is a special transaction that may create new bitcoins and colelcts the transactions fees
  • Target: corresponds to the difficulty of finding a new block. Updated every 2016 blocks when the difficulty reset occurs
  • The Block's Hash: all of the above items get hashedd into the block hash which, for one, is proof that the other parts of the header have not been changed, and then is used as a reference by the succeeding block
  • Nonce: an arbitrarily picked number to conveniently add entropy to a block header without rebuilding the Merkle tree

    Blockchains!

    Many miners compete to find the next block. Often, there will be more than one valid block discovered, but if there is a fork, only one progresses to a greater length and the shorter is discarded. The discarded blocks are called extinct or orphaned blocks.

    Things to Note About the Blockchain!

    1. Everyone can start from the Genesis Block and apply each block consecutively to arrive at the same result.
    2. When you have applied the latest block, the balances in your ledger have the exact same state as in all the other participants' ledgers.
    3. As each block builds upon its predecessor, each new block buries the history under more work
    4. Pretty safe!
      To change the ledger, you not only would have to harness a ton of computer power, but you'd also have to do it in very public space where thousands of other computers and users can see exactly what you're doing.
      What one person or computer does affects the entire blockchain, and everyone can police the transactions.
